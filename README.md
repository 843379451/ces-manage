### 简介  
$ 中后台管理系统模版，基于koa+vue  
### 在线演示地址  

### 后端koa-ces  
$ 安装依赖 npm i  
$ 启动 npm run start  
$ [接口说明](https://github.com/XingGuoZM/ces-manage/blob/master/koa-ces/README.md)  

### 前端vue-ces  
$ 安装依赖 npm i  
$ 启动 npm run dev 或者 npm run start  
$ 打包构建 npm run build  
$ [前端说明](https://github.com/XingGuoZM/ces-manage/blob/master/vue-ces/README.md)
